# Specs

Этот каталог фиксирует целевую постановку проекта и служит опорой для команды при реализации симулятора.

Состав:

- `vision.md` — цель работы, ограничения и ключевая идея модели.
- `architecture.md` — целевая архитектура симулятора и pipeline обработки.
- `data-model.md` — основные сущности и форматы данных.
- `spectrometer-integration.md` — как подружить симулятор с реальным спектрометром.
- `team-plan.md` — распределение обязанностей на команду из 5 человек.
- `mvp-roadmap.md` — поэтапный план сборки первой рабочей версии.

Главная инженерная ставка проекта:

- центральной сущностью становится спектральная карта сцены;
- моделирование строится на массивах интенсивностей и коэффициентах отражения, а не на поштучной трассировке фотонов;
- оптика, сенсор и цифровая обработка разделяются на независимые слои.
