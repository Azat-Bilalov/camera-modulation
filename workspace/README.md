# Workspace

Этот каталог организует изолированные рабочие места по ролям из `specs/team-plan.md`.

Структура:

- `models.py` — общие модели данных для всех участников.
- `shared.py` — общие утилиты и детерминированные значения по умолчанию.
- `main.py` — точка интеграции для Азата Билалова (архитектора).
- `scene_source/` — рабочее место Максима Гостева (модель сцены и источника).
- `optics/` — рабочее место Егора Андронаки (оптический тракт).
- `sensor_adc/` — рабочее место Эрнеста Литвиненко (сенсор, АЦП и калибровка).
- `visualization/` — рабочее место Егора Лиличева (визуализация, экспорт и верификация).

Принцип:

- каждый работает только в своей подпапке;
- каждая функция-заглушка возвращает корректный объект общей модели;
- архитектор собирает все роли в `workspace/main.py`.

Запуск интеграции:

```bash
python3 -m workspace.main
```
