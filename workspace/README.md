# Workspace

Этот каталог организует изолированные рабочие места по ролям из `specs/team-plan.md`.

Структура:

- `models.py` — общие модели данных для всех участников.
- `shared.py` — общие утилиты и детерминированные значения по умолчанию.
- `main.py` — точка интеграции для архитектора.
- `scene_source/` — рабочее место роли "модель сцены и источника".
- `optics/` — рабочее место роли "оптический тракт".
- `sensor_adc/` — рабочее место роли "сенсор, АЦП и калибровка".
- `visualization/` — рабочее место роли "визуализация, экспорт и верификация".

Принцип:

- каждая роль работает только в своей подпапке;
- каждая функция-заглушка возвращает корректный объект общей модели;
- в docstring каждой функции указано, что подается на вход, что ожидается на выходе и где смотреть пример в `draft/`;
- архитектор собирает все роли в `workspace/main.py`.

Запуск интеграции:

```bash
python3 -m workspace.main
```
