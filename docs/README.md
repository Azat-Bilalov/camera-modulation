# Документация — Отчёт по проекту «Моделирование камеры»

Документ подготовлен в рамках курса «Техническое зрение» (кафедра «Прикладная информатика», Московский Политех) для фиксации прогресса по проекту спектральной камеры с кодирующей маской.

## Структура

```
docs/
├── main.typ                  # Точка входа: стили, шрифты, подключение глав
├── bibliography.bib          # Библиография (BibTeX)
├── gost.csl                  # Стиль цитирования ГОСТ
├── main.pdf                  # Скомпилированный документ
│
├── chapters/                 # Главы документа
│   ├── титульный_лист.typ    # Титульный лист (ГОСТ оформление)
│   ├── introduction.typ      # Введение / цель отчёта
│   ├── теория.typ            # Описание архитектуры и прогресса
│   ├── заключение.typ        # Заключение
│   └── список_литературы.typ # Список литературы
│
├── addons/                   # Пользовательские модули Typst
│   ├── cyrillic_enum.typ     # Кириллическая нумерация списков
│   ├── bib_count.typ         # Подсчёт цитирований
│   ├── tables.typ            # Хелперы для таблиц
│   └── variables.typ         # Переменные задач проекта
│
├── assets/                   # Изображения и медиа-ресурсы
├── figures/                  # Рисунки (авто-нумерация)
├── images/                   # Дополнительные изображения
├── tables/                   # Файлы таблиц
└── word/                     # Исходные материалы (docx)
```

## Сборка

Для компиляции документа требуется [Typst](https://typst.app):

```bash
# Установка Typst (macOS)
brew install typst

# Сборка PDF
typst compile docs/main.typ docs/main.pdf

# Режим наблюдения (авто-пересборка при изменениях)
typst watch docs/main.typ docs/main.pdf
```

## Зависимости

Документ использует следующие пакеты Typst (загружаются автоматически из `@preview`):

| Пакет | Версия | Назначение |
|-------|--------|------------|
| `muchpdf` | 0.1.1 | Работа с PDF-вставками |
| `zebraw` | 0.5.5 | Подсветка блоков кода |

## Оформление

Документ настроен по требованиям ГОСТ:

- Формат: **A4**, поля — лево 30 мм, право 15 мм, верх/низ 20 мм
- Шрифт: **Times New Roman**, 14 pt, межстрочный интервал — базовый
- Абзацный отступ: **1.25 см**
- Нумерация заголовков: **1.1**, **1.1.1** (до 3 уровня)
- Рисунки: подпись «Рисунок», нумерация привязана к главе
- Цитирование: **ГОСТ** (автоматический подсчёт через `bib_count.typ`)
