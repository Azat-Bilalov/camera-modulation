# Streamlit-обёртка над Camera Modulation Pipeline

Это интерактивный GUI, который использует готовую логику из `workspace/` и `draft/models.py`, не внося изменений в исходные модули проекта.

## Запуск

```bash
source .venv/bin/activate
streamlit run streamlit_app/app.py
```

После запуска приложение будет доступно по адресу `http://localhost:8501`.

## Возможности

- **Боковая панель** — настройка всех ключевых параметров симуляции:
  - Спектральная ось (диапазон и шаг)
  - Источник света (интенсивность, центр и ширина спектра)
  - Размер сцены
  - Пропускание оптических каналов
  - Параметры сенсора (gain, dark offset, время экспозиции)
  - Параметры АЦП (разрядность, full scale)

- **Визуализация результатов**:
  - Итоговое grayscale-изображение после реконструкции
  - Промежуточные карты: экспозиция, заряд, оптические каналы LOW/HIGH
  - График спектра источника
  - Гистограмма распределения яркости цифрового кадра

- **Верификация**:
  - Статистика кадра (размер, динамический диапазон, среднее)
  - Проверка диапазона АЦП
  - Анализ клиппинга с визуальными предупреждениями

## Архитектура

`streamlit_app/app.py` напрямую импортирует:
- `workspace.models` — общие dataclass-модели
- `workspace.shared` — расчётные функции (спектры, экспозиция, заряд, квантование)
- `workspace.visualization.verifier` — проверки качества кадра

Таким образом, вся физика остаётся в `workspace/`, а `streamlit_app/` отвечает только за UI.
