# Draft

Этот каталог содержит черновые runnable-примеры по этапам будущего pipeline.

Структура:

- `00_dataclasses/` — демонстрация dataclass-сущностей и их ролей в pipeline.
- `models.py` — подробно подписанные dataclass-модели проекта.
- `lib.py` — общие структуры данных и базовые функции симуляции.
- `01_spectral_axis/` — пример задания спектральной сетки.
- `02_scene/` — генерация спектральной карты сцены.
- `03_optics/` — упрощенный оптический тракт.
- `04_sensor/` — накопление сигнала на сенсоре.
- `05_adc/` — квантование через АЦП.
- `06_reconstruction_export/` — реконструкция кадра и экспорт `bmp`.
- `07_full_pipeline/` — итоговый рабочий сквозной пример.

Запускать можно так:

```bash
python3 draft/00_dataclasses/example.py
python3 draft/01_spectral_axis/example.py
python3 draft/07_full_pipeline/main.py
```

Принцип этого набора:

- каждая подпапка показывает отдельный этап;
- последний этап собирает все в единый сценарий;
- используется упрощенная физика, но правильная структура данных;
- примеры не требуют внешних библиотек.
